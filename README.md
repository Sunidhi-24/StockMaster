
# Stock Master

Stock Master is a comprehensive warehouse and stock management web application built with modern web technologies. It provides a user-friendly interface for managing inventory, tracking stock movements, handling receipts and deliveries, and overseeing warehouse operations.

## Features

- **Authentication**: Secure login and signup with demo credentials (Login ID: admin, Password: Password123!)
- **Dashboard**: Overview of key metrics and quick access to main functions
- **Stock Management**: View and manage stock levels, SKUs, and low stock alerts
- **Operations**: Handle warehouse operations and processes
- **Move History**: Track historical stock movements
- **Receipts & Deliveries**: Manage incoming receipts and outgoing deliveries with detailed views
- **Stock Ledger**: Maintain a ledger of all stock transactions
- **Settings**: Configure application preferences
- **Warehouse & Location Management**: Organize inventory by warehouse and location

## Technologies Used

- **Frontend Framework**: React 18 with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS with custom theme variables for light/dark modes
- **UI Components**: shadcn/ui (built on Radix UI primitives) for accessible, customizable components
- **Icons**: Lucide React
- **Charts**: Recharts for data visualization
- **Forms**: React Hook Form with validation
- **Notifications**: Sonner for toast notifications
- **Other Libraries**: Embla Carousel, Input OTP, Vaul (drawer), etc.

## Running the Code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.
